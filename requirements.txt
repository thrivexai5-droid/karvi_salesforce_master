# Installation Instructions for PythonAnywhere:
# 1. Upload your project files to PythonAnywhere
# 2. Open a Bash console on PythonAnywhere
# 3. Create virtual environment: python3.10 -m venv venv
# 4. Activate virtual environment: source venv/bin/activate
# 5. Install requirements: pip install -r requirements.txt
# 6. Set up PostgreSQL database in PythonAnywhere dashboard
# 7. Update .env file with PythonAnywhere database credentials
# 8. Run migrations: python manage.py migrate
# 9. Collect static files: python manage.py collectstatic
# 10. Create superuser: python manage.py createsuperuser
# 11. Configure WSGI file in Web tab

# PythonAnywhere Specific Notes:
# - Use PostgreSQL database (included in paid plans)
# - Static files are handled by WhiteNoise
# - Make sure to set DEBUG=False in production
# - Update ALLOWED_HOSTS with your PythonAnywhere domain
